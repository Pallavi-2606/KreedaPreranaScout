# Kreeda Prerana Scout

Kreeda Prerana Scout is an Android application developed using Kotlin for athlete management and performance tracking.

## Features
- Athlete Management
- Search Athletes
- Trial Performance Tracking
- Leaderboard
- Analytics Dashboard
- Firebase Integration
- Room Database

## Tech Stack
- Kotlin
- Android Studio
- Firebase Firestore
- Room Database
- RecyclerView
- Material UI
